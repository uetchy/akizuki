# Akizuki

This gum is currently extremely unstable. Don't use for production environment.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'akizuki'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install akizuki

## Usage

```ruby
Akizuki.find "Arduino"
```
